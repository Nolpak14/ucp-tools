## IMAGE DESIGNS NEEDED

### 1. Profile Logo/Avatar (400x400px)
**File:** `social/logo-400x400.png`

**Design:**
- Use existing logo from `public/logo.jpeg`
- Shopping bag with checkmark on gradient background
- Gradient: `#2E86AB` ‚Üí `#36B5A2` ‚Üí `#47C97A` (135deg)
- Ensure checkmark is clearly visible at small sizes
- Add subtle shadow for depth
- **Important for X:** Center key elements (corners get cropped to circle)

### 2. LinkedIn Company Banner (1128x191px)
**File:** `social/linkedin-banner-1128x191.png`

**Design Elements:**
- Left side: Logo icon (small, ~80px)
- Center: "UCP.tools" text with gradient
- Right side: Tagline "Get ready for AI commerce"
- Background: Light gradient or solid `#F8FAFC` with subtle pattern
- Brand gradient accent line at bottom

**Text Content:**
```
[Logo] UCP.tools | Get ready for AI commerce
```

### 3. X.com Header Banner (1500x500px)
**File:** `social/x-header-1500x500.png`

**Design Elements:**
- Logo icon on left (avoid bottom-left 200x200px area - profile picture overlaps)
- "UCP.tools" text prominent in center
- Tagline: "Validate ‚Ä¢ Generate ‚Ä¢ Ship"
- Background: Brand gradient with subtle commerce/tech pattern
- Safe zone: Keep important content in center 1100x400px area

**Text Content:**
```
UCP.tools
Validate ‚Ä¢ Generate ‚Ä¢ Ship
Free AI Commerce Readiness Checker
```

### 4. General Social Post Image (1200x630px)
**File:** `social/og-social-1200x630.png`

**Design:**
- Can reuse/adapt existing `public/og-image.png`
- Logo + tagline + key value proposition
- Works for both LinkedIn posts and X.com link previews
