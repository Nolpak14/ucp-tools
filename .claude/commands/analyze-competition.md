# Competition Analysis

Run a comprehensive competition analysis for UCPtools to identify new competitors, tools, and market changes in the UCP ecosystem.

## Instructions

1. Search the web for new UCP-related tools and services using these queries:
   - "UCP validator tool"
   - "universal commerce protocol tools"
   - "UCP compliance checker"
   - "agentic commerce tools 2026"

2. Check GitHub for new/trending repositories:
   - Search: "universal commerce protocol"
   - Search: "ucp validator"
   - Search: "agentic commerce"

3. Monitor key competitors:
   - Google Merchant Center UCP features
   - Shopify UCP documentation updates
   - Any new startups in the space

4. Check for market signals:
   - New funding announcements in agentic commerce
   - Platform announcements (Shopify, WooCommerce, BigCommerce)
   - Developer community activity

5. Generate a report with:
   - New competitors found (if any)
   - Trending GitHub repos
   - Market signals and news
   - Recommendations for UCPtools positioning

6. Save the report to `docs-private/competition-reports/` with today's date.

## Output Format

Provide a summary with:
- **Threat Level**: Low/Medium/High
- **New Competitors**: List any new tools found
- **Market Trends**: Key observations
- **Action Items**: What we should do in response
